# MaterialUp-Client-for-Android
A simple MaterialUp Client for Android

This is a "Scratch-Your-Own-Itch" project.
MaterialUp is insanely awesome and I often want to use it on my Android device

This is an Open Source Android Client for MaterialUp made using 
this [Unofficial MaterialUp Wrapper Library](https://github.com/Alelak/materialup "materialup") 
by [Alladin El Akhrass](https://github.com/Alelak).

Feel free to contribute to make the project better!
